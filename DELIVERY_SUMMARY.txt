HOW TO USE
================================================================================

STEP 1: Install Requirements
   pip install pandas numpy matplotlib openpyxl statsmodels

STEP 2: Run the GUI Application
   python gui_config.py

STEP 3: Configure Data Sources
   - Click "+ Add Data Source"
   - Click "Browse" to select Excel file
   - Location auto-fills from filename (edit if needed)
   - Specify DateTime column index
   - Specify Temperature column index
   - Optionally specify Lux column (or -1 to skip)
   - Click "Preview" to verify data
   - Repeat for each data source

   - Click "Browse" under Ship Schedule Data
   - Select Excel file with ETA/ETD information
