## üöÄ Getting Started in 3 Steps

```bash

```

### Step 2: Run the GUI

### Step 3: Configure and Analyze

- Click Preview to verify columns
- Select analysis options
- Click "Start Analysis"

**That's it!** Results appear automatically.
